# ShramSafal Marketing Site — Imagen 3 Prompt Sheet

## Global Style Brief (paste this before EVERY prompt)

```
Style: Warm editorial illustration, semi-realistic, textured watercolor-meets-digital.
NOT cartoon, NOT 3D render, NOT stock photo. Think "Kurzgesagt meets rural India editorial".
Color palette: emerald green (#059669), warm amber (#D89B1D), earth brown (#78350F),
cotton white (#EDE8DF), soil umber (#5C4A3A), soft cream (#FFFDF8).
Cultural context: Maharashtrian farmers — men in white cotton kurta, turban (pheta/pagdi),
dhoti or trousers. Dark brown skin (#A67B5B to #8B6545). Crops: grapes on trellis,
pomegranate trees, sugarcane fields, onion plots. Landscape: Deccan plateau, red-brown soil,
open sky. Phone: basic Android smartphone with green UI.
Mood: Dignified, warm, hopeful — never pitiful or condescending.
Output: PNG, transparent or cream (#FFFDF8) background, 2400px wide, clean edges.
No text/watermarks/logos in the image.
```

### Font Overlay Rules (for any text added in code, NOT in images)
- Marathi body: `'Noto Sans Devanagari', sans-serif`
- Marathi decorative: `'Noto Serif Devanagari', serif`
- English/brand: `'DM Sans', sans-serif`
- NEVER bake text into generated images — all text stays in HTML/CSS

### Integration Priority
1. **High impact** (generate first): Hero, Before/After pair, Legacy, CTA
2. **Medium impact**: Problem, Identity, Farmer-Questions, Partnership
3. **Atmospheric** (generate last): QA backgrounds (x5), Workflow, Clarity, Trust, Value

### Performance Notes
- Convert all PNGs to WebP after generation (80% quality)
- Use `<img loading="lazy">` for below-fold images
- Hero image: eager load, add `fetchpriority="high"`
- QA backgrounds: use as CSS `background-image` at 4-8% opacity
- All images go in `public/images/generated/` folder
